# LG-S42-002: n8n-mcp Update Available

**Severity:** LOW
**Status:** OPEN
**Session:** 42
**Date:** 2026-03-31

## Description

The n8n-mcp integration is running version 2.42.3. Version 2.44.0 is available.

## Impact

- May miss bug fixes or new node types in the newer version
- No known breaking issues with current version

## Recommended Fix

1. Stop n8n (`n8n stop` or kill process)
2. Run `npm update -g n8n` or `npm install -g n8n@latest`
3. Restart n8n and verify healthz endpoint
4. Re-test existing workflows

## Workaround

Current version (2.42.3) is functional. Update can be deferred to a maintenance window.
