# User Apps — The Workhorse

Pre-configured tools ready to run. Double-click any `.cmd` file or run the `.py` scripts directly.

## Available Apps

| App | File | What It Does |
|-----|------|-------------|
| **Phone Check** | `phone-check.cmd` | Plug in a phone via USB → auto-scan for malware/spyware |
| **Photo Timeline** | `photo-timeline.cmd` | Point at a folder of photos → generates a vacation timeline |
| **Voice Clone** | `voice-clone.cmd` | Clone a voice from audio samples → generate speech in that voice |
| **Avatar Creator** | `avatar-creator.cmd` | Create a talking-head video from a photo + audio |
| **Quick Scan** | `quick-scan.cmd` | Fast security check of the PC (services, ports, processes) |
| **Market Check** | `market-check.cmd` | Current prices for your watchlist + forex rates |

## Usage

All apps are self-contained. They import from the `rudy/` package on the Desktop.
Results are saved to `rudy-logs/` or `rudy-data/` and printed to the console.
